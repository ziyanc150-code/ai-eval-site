多模态 AI 自动化评测平台

上传数据文件后进行批量评测。必须使用你自己的「评测 API Endpoint」与「API Key」，请求由浏览器直接发往你的网关；本站不会、也不可用站长的密钥代付，避免产生站长的模型费用。

请务必为你的 API 配置 CORS，允许当前站点来源（浏览器地址栏的 origin），否则浏览器会拦截跨域请求。

1) 基本设置

2) 评测模板与自定义维度

每个维度需填写独立评测提示词；提示词中可使用占位符 {{item}}（当前样本 JSON）、{{dimension_name}}、{{criteria}}、{{task_type}}、{{dimensions}}（当前维度一行说明）。模型需按该维度返回 scores 中含对应维度名的分数。

3) 从飞书导入（可选）

支持 多维表格（Bitable）/ 电子表格（Sheet）/ 云空间文件（Drive，xlsx/图/视频）。App ID/Secret 仅存于你的浏览器 localStorage，不上传本站数据库；请求通过本站 Cloudflare Function 转发飞书接口（必须，因为飞书不支持浏览器直连）。

4) 上传数据

可任选一种或两者结合：
• 结构化数据文件（可选）：JSON / JSONL / XLSX（第一行为表头），字段可包含 id、input_text、output_text、image_url、video_url、frame_urls、file（媒体文件名，用于匹配）。
• 媒体文件（可选，多选）：直接上传图片 / 视频。未提供数据文件时，每个媒体文件会成为一条样本；提供数据文件时，按行中 file 或 id 与文件名匹配，自动注入为 image_url / video_url（Base64 Data URL）。
• 浏览器会把媒体读成 Base64 并随请求直传你的 API，单文件建议 ≤ 8 MB（视频尤需压缩），否则可能触发模型上下文或 API 限制。
